# mediumi: A Media Toolkit

## Build
- Build all
```
$ make build-all
```
- Build specific crates
```
$ make build-aac
$ make build-ac3
$ make build-h264
$ make build-mpeg2ts
...
```

## Test
- Test all
```
$ make test-all
```
- Test specific crates
```
$ make test-aac
$ make test-ac3
$ make test-h264
$ make test-mpeg2ts
...
```

## Crates
- mediumi-aac
    - AAC(ADTS)
- mediumi-ac3
    - AC-3
- mediumi-h264
    - H.264(AVC)
- mediumi-mpeg2ts
    - MPEG2-TS encoder/decoder
- ...
