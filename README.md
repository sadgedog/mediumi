# mediumi: A Media Toolkit

## Build
- Build all
```
$ make build-all
```
- Build specific crates
```
$ make build-mpeg2ts
$ make build-codec
...
```

## Test
- Test all
```
$ make test-all
```
- Test specific crates
```
$ make test-mpeg2ts
$ make test-codec
...
```

## Crates
- mediumi-mpeg2ts
    - MPEG2-TS encoder/decoder
- mediumi-codecs
    - H.264(AVC)
    - AAC(ADTS)
- ...
